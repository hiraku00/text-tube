# Rules

このフォルダには、AI が開発を行う際に遵守すべき「プロジェクトの絶対ルール」が格納されています。

## 仕組み
1. **ルールの定義**: 各 `.md` ファイル（例: `security.md`, `coding-style.md`）に、具体的なガイドラインを記述します。
2. **ルールの適用**: Antigravity の `.agent/skills/rules-consultant/` スキルが、このフォルダ内のファイルを読み込みます。
3. **運用**: エージェントに対して「ルールを確認して」や「ルールに違反していないかチェックして」と指示することで、ここに書かれた内容に基づいた回答やコード生成が行われます。

## ファイル構成
- `security.md`: セキュリティに関する必読事項（APIキーの扱いや入力検証など）。
- `coding-style.md`: コードの書き方、命名規則、ディレクトリ構造などの規約。
- `testing.md`: テストの方針、カバレッジ基準など。
