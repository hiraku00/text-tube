# Rules Consultant Skill

このスキルは、エージェントがプロジェクト固有のルールやガイドラインを遵守するために使用されます。

## 役割
- `rules/` ディレクトリ内のすべての `.md` ファイルを読み込み、そこに記載されたガイドラインに従って行動します。
- コードスタイル、セキュリティ、テスト、Git ワークフローなどのルールを適用します。

## 手順
1. `rules/` 内のファイル一覧を確認します。
2. 関連するルールファイルを読み込みます。
3. ユーザーの要求がルールに適合しているか検証し、必要に応じて修正を提案します。
